# 🏋️ AI Fitness Coach Agent

An intelligent Discord bot powered by AI that generates personalized workouts, tracks your fitness progress, and provides real-time coaching feedback. Built with Groq's Llama 3.1 LLM and Supabase for persistent data storage.

[![Discord](https://img.shields.io/badge/Discord-7289DA?style=flat&logo=discord&logoColor=white)](https://discord.com/)
[![Node.js](https://img.shields.io/badge/Node.js-339933?style=flat&logo=node.js&logoColor=white)](https://nodejs.org/)
[![Supabase](https://img.shields.io/badge/Supabase-181818?style=flat&logo=supabase&logoColor=white)](https://supabase.com/)

## 🎯 Features

### ✅ Implemented
- **Personalized Workout Generation** - AI creates workouts based on your fitness level, goals, and injury history
- **Progress Tracking** - Log exercises, sets, reps, and weight lifted
- **Persistent Memory** - Your preferences and workout history saved in Supabase
- **AI Coaching** - Real-time feedback and motivational coaching after each workout
- **Fitness Analytics** - View statistics on total volume, unique exercises, and progress
- **Discord Integration** - Simple slash commands for easy access
